# INACOMP 2.0 (Informatics National Competition)

Official landing page for **INACOMP 2.0**, a national informatics competition organized by the Student Representative Body (BEM) of Informatics and Computer Engineering Education, Jakarta State University (UNJ).

This project features a modern, responsive, and high-performance landing page designed for high school and vocational students (SMA/SMK/Sederajat) across Indonesia.

## 🏆 Competition Tracks

- **Competitive Programming**: Technical and contest-oriented platform for algorithm enthusiasts.
- **Cisco Networking**: Enterprise-grade networking infrastructure competition.
- **UI/UX Design**: Modern and creative design track for future designers.

## 🚀 Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## ✨ Key Features

- **Modern UI/UX**: Clean, polished, and youthful design system.
- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile devices.
- **Fast Performance**: Optimized using Next.js latest features.
- **SEO Optimized**: Meta tags and structured data for better search engine visibility.
- **Interactive Elements**: Smooth animations and transitions using Framer Motion.

## 🛠️ Getting Started

### Prerequisites

- Node.js 20+
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/landing-page.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📂 Project Structure

- `src/app`: Next.js App Router pages and layouts.
- `src/components`: Reusable UI components and section layouts.
- `src/lib`: Utility functions and shared configurations.
- `public/assets`: Static assets including images and brand logos.
