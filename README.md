Saya ingin kamu bertindak sebagai senior UI engineer + frontend engineer yang
ahli dalam membuat landing page modern, responsive, dan konsisten secara visual.

Konteks proyek: Saya sedang membuat website untuk INACOMP 2.0 (Informatics
National Competition 2.0), yaitu kompetisi nasional di bidang:

1. Competitive Programming
2. Cisco Networking

INACOMP diselenggarakan oleh Badan Eksekutif Mahasiswa Prodi Pendidikan Teknik
Informatika dan Komputer Universitas Negeri Jakarta, dan ditujukan untuk siswa/i
SMA/SMK/sederajat.

Tujuan: Saya ingin membuat website landing page yang terasa modern, profesional,
tech-oriented, youthful, dan meyakinkan. Website ini akan terdiri dari 3 halaman
utama yang dikerjakan pada 3 session terpisah:

- Home / General Page
- Competitive Programming Page
- Cisco Networking Page

Semua halaman WAJIB memiliki style visual yang sama agar terasa satu brand,
namun masing-masing halaman tetap punya karakter section dan konten yang
berbeda.

Arah desain utama yang wajib diikuti:

1. Gaya visual utama mengacu pada nuansa modern, rounded, clean, polished, dan
   youthful seperti website brand startup modern.
2. Untuk halaman home, gunakan nuansa visual yang bold, premium, engaging,
   colorful tetapi tetap rapi.
3. Untuk halaman Competitive Programming, gunakan nuansa yang lebih clean,
   structured, technical, dan contest-oriented.
4. Untuk halaman Cisco Networking, gunakan nuansa yang lebih enterprise-tech,
   network-oriented, dengan aksen visual yang terinspirasi dari dunia
   Cisco/network infrastructure.
5. Semua halaman harus tetap terasa berasal dari 1 design system yang sama.

Design system yang harus dipakai:

- Gunakan style modern, polished, clean
- Border radius besar dan konsisten
- Card dengan border tipis dan shadow halus
- Layout lega, whitespace cukup
- Typography modern, tegas, mudah dibaca
- CTA jelas dan menonjol
- Komponen terasa premium tetapi tidak berlebihan
- Hindari desain terlalu ramai, terlalu childish, atau terlalu corporate kaku

Warna dan mood:

- Basis brand utama: biru / indigo / navy modern
- Home: lebih bright, energetic, modern
- Competitive Programming: clean, putih / slate terang dengan aksen biru,
  emerald, amber
- Cisco Networking: navy / blue / cyan / teal, lebih tech-infrastructure feel
- Pastikan warna antar halaman tetap harmonis dan satu keluarga brand

Typography:

- Gunakan nuansa font modern seperti Sora / Inter / Plus Jakarta Sans / Space
  Grotesk
- Heading harus kuat dan modern
- Body harus sangat readable
- Buat hierarchy heading, subheading, label, dan body dengan jelas

Komponen visual:

- Gunakan card modern
- Gunakan badge kecil bila perlu
- Gunakan section divider halus
- Gunakan skeleton / placeholder untuk gambar, dokumentasi, dan media
- Jangan gunakan gambar final dari internet
- Untuk seluruh area galeri / media / thumbnail, gunakan skeleton atau
  placeholder box yang rapi
- Untuk ikon, gunakan ikon modern yang sederhana dan konsisten

Responsive wajib:

- Website WAJIB responsive
- Harus bagus di desktop, tablet, dan mobile
- Perhatikan stacking layout di mobile
- Pastikan tombol, card, timeline, galeri, FAQ, dan navbar tetap nyaman
  digunakan di layar kecil
- Jangan hanya desktop-first; implementasi harus benar-benar mobile-friendly

Kebutuhan implementasi:

- Bangun halaman dengan struktur yang rapi dan reusable
- Jika menggunakan React/Next.js, buat komponen section yang bersih dan mudah
  dikembangkan
- Gunakan Tailwind CSS
- Pastikan spacing, sizing, typography, dan responsive breakpoint tertata rapi
- Gunakan semantic HTML yang baik
- Fokus pada kualitas UI dan struktur layout
- Gunakan placeholder/skeleton untuk gambar
- Jangan isi dengan asset image nyata

Output yang saya inginkan:

- Hasil implementasi halaman yang polished dan siap dikembangkan lebih lanjut
- Konsisten dengan design system global
- Responsive
- Clean code structure
- Komponen reusable
- Tidak perlu backend
- Tidak perlu fitur kompleks
- Fokus pada landing page UI/UX

Penting:

- Jangan membuat style yang berbeda jauh antar halaman
- Semua halaman harus terasa satu website / satu brand
- Yang berbeda hanya penekanan visual sesuai cabang lomba
- Gunakan skeleton untuk seluruh image area
- Jika ada section galeri, tampilkan dalam bentuk placeholder cards/skeleton
  gallery
- Jika ada area download guidebook, tampilkan tombol CTA yang menonjol
- Jika ada timeline, buat desain timeline yang unik dan modern, bukan timeline
  garis biasa
- FAQ sebaiknya pakai accordion yang responsive
- Hasil akhir harus tampak realistis seperti landing page event teknologi
  nasional

Saat mengerjakan halaman, lakukan hal berikut:

1. Pahami dulu konteks halaman yang diminta
2. Buat struktur section yang sesuai
3. Implementasikan UI dengan konsisten terhadap global design system di atas
4. Pastikan responsive
5. Pastikan semua placeholder/skeleton sudah rapi
6. Jangan keluar dari style global
