# DESIGN.md — INACOMP 2.0 Landing Page

## Project Overview

INACOMP (Informatics National Competition) is a national-scale IT competition for Indonesian high school students (SMA/SMK sederajat).

This website should feel:
- modern
- academic
- competitive
- youthful
- national-scale
- premium but approachable

The design MUST NOT feel like:
- AI SaaS startup
- crypto/web3 landing page
- cyberpunk dashboard
- generic AI-generated template
- hacker movie aesthetic

The website should communicate:
“serious national IT competition with modern youth branding.”
